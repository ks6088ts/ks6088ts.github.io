title: ノート PC の買い替えを検討
date: "2020-11-06T18:56:27.000Z"
description: "MacBookPro か Windows ノートか"
---

仕事では Windows、個人開発用には、MacBook Pro を使い続けているが、そろそろノートPCを買い替えたくなってきた。  
そもそも Mac を使っている理由は、当時の Windows での開発環境がイケてなかったからなのだが、  
最近の Windows は WSL2 や Windows Terminal のおかげでほぼ Ubuntu な開発環境が構築できる。  
WSL2 内で完結した開発をする限りにおいては、もはや Windows でも良いのではないか。  
MacBook Pro で Windows 相当のスペックを要求すると 20 万はあっさり超えてしまうし、  
TouchBar が個人的に好きじゃないし、Windows PC 安いし、、  
Windows ノート PC を買いましょうか。。
