Unity に今更ながら入門した。  
Web 上に良質な記事を見つけたので 2 時間ほどで基礎はキャッチアップできた。  
C# スクリプトの実装に関してはドキュメントを見れば判るし、あとはそれをどのオブジェクトにどうアタッチするかを軽く考えればやりたいことはできるようになりそう。  

# Unity 学習フロー

Unity に一切触ったことがない状態からのスタート。  
自分がやった作業としては、

1. Mac 上に [Unity Personal](https://store.unity.com/ja/products/unity-personal) (無料版) をインストール
2. [今日からはじめるUnity](https://qiita.com/nmxi/items/7950fb12ef925efa276d) をなぞる

ぐらいで、ブロック崩しまでは Unity で動かせるようになった。  
Unity って GUI 操作結構多くなるので GIF アニメで解説あってとても助かる。  
そしてその操作方法を知っているだけで大分作業効率に差が出る。
