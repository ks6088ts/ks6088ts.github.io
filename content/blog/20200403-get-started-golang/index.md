# Go 言語の特徴

* 静的型付け: コンパイル時エラー検出、静的解析
* 並行プログラミング: goルーチン, chan によるデータやり取り
* 周辺ツール: go build / go test / go doc / gofmt, goimports / go vet, golint
* シングルバイナリ・クロスコンパイル: 環境変数 GOOS, GOARCH。動作環境を用意する必要がない

**利用事例**

* OSS: Docker, Kubernetes, gVisor
* CLI: peco, ghq, Hugo
* IoT: gobot, 
* サーバ: gRPC
* ツール: [reviewdog](https://github.com/reviewdog/reviewdog)
